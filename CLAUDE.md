# SoupOfTheDay
# Project Overview
This is a GUI based program designed to calculate the best elixir and meal recipes to make in Legend of Zelda Tears of the Kingdom.  It needs to have fast smooth input for building recipes, with visual buttons AND a text search field, along with filters for different material types (fruits, moster parts, guts, minerals, ingredients etc)

# Architecture
This will have a web front-end built in html & CSS, back end is tbd

# Recipe sell value formula
Final sell value for an elixir or recipe is calculated as follows:
Sum of the sell value for all components, multiplied by a value based on total number of ingredients:
1 item - 1.2x
2 items - 1.3x
3 items - 1.4x
4 items - 1.6x
5 items - 1.8x
